# Pakistan Poverty Line 2025 — Is Rs. 8,483 Enough?

The government's Economic Survey 2024-25 set Rs. 8,483 per month as the national poverty line — meaning anyone earning above this is officially "not poor." This project breaks down what survival actually costs across 5 Pakistani cities and compares it against that threshold.

## Key Finding
In every single city — Karachi, Lahore, Islamabad/Rawalpindi, Peshawar, and interior Sindh — the real minimum survival cost exceeds the official poverty line by 2x to 5x.
